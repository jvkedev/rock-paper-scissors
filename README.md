# Rock Paper Scissors

This is a simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.
You play against the computer, and the game instantly shows who wins each round.
The interface is clean, responsive, and easy to use.

## Features

- Player vs Computer mode
- Shows win, lose, or draw after every round
- Score tracking
- Simple and clean UI
- Responsive layout for mobile and desktop

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**

## How to Run

Download or clone the project:

```bash
git clone https://github.com/jvkedev/rock-paper-scissors.git
```

Open index.html in your browser.

## Notes

- This is a static frontend project.
- No backend, database, authentication or API calls are used.
- Perfect for beginners practicing DOM manipulation and game logic.
